# to.software_function
software_function triggers a function that is implemented by software.
- cg_event_double_click Send double click event by software.
- iokit_power_management_sleep_system Causes an immediate system sleep.
- open_application Open an application, or if it’s already running, bring it into focus.
- set_mouse_cursor_position Move the mouse cursor to the specified point.
