# CSS
Zed has built-in support for CSS.
- Tree-sitter: tree-sitter/tree-sitter-css
- Language Servers: microsoft/vscode-css-languageservice tailwindcss-language-server
## Tailwind CSS
Zed also supports Tailwind CSS out-of-the-box for languages and frameworks like JavaScript, Astro, Svelte, and more.
